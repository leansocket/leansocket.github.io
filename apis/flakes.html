# Flake

雪花算法唯一编码生成器

  • 此类采用flake算法生成唯一编码。
  • 此类能同时支持156个节点，每个节点每毫秒生成2^24个唯一编码。
  • 此类生成的唯一编码是一个由20个十六进制字符组成的串。

Kind: global class
