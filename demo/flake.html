# 唯一编码生成器

框架提供基于flake的唯一编码生成方法,可以在往数据库里添加数据时，用于生成唯一的id号。

如下使用flakes.create生成10个不重复的id。

const libs = require('nodex-libs');
const flakes = libs.flakes;

let f = flakes.create();
for(let i = 0; i < 10; i++){
    console.log(f.get());
}
